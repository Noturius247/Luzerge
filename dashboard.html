    Ready to submit for CDN setup?

    Choose your CDN provider. Cloudflare offers the most features; other providers support analytics, DNS, SSL & uptime monitoring.

    Your domain can be on any hosting provider (GoDaddy, Hostinger, AWS, VPS, etc.). Cloudflare sits in front as a reverse proxy — your origin server stays the same.

    1. Create a free Cloudflare account Go to dash.cloudflare.com/sign-up and sign up
    2. Add your domain Click "Add a Site", enter your domain, and select the Free plan
    3. Change your nameservers Cloudflare will give you two nameservers (e.g. anna.ns.cloudflare.com). Update these at your domain registrar (GoDaddy, Namecheap, etc.). This usually takes 5–30 minutes to propagate.
    4. Get your Zone ID In Cloudflare dashboard → your domain → Overview → scroll down to the API section on the right sidebar. Copy the Zone ID.
    5. Create an API Token Go to My Profile → API Tokens → Create Token → use the "Edit zone DNS" template → under Zone Resources, select your domain → Continue to summary → Create Token. Copy the token.
    6. Paste below Enter both values in the fields below and submit. All panels will show live data instantly.
    Your credentials are stored securely and only used to communicate with your CDN's API on your behalf. We never share them.
