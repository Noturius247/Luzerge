    Configure Domain

    How to get these credentials

    1 Get the Zone ID

    1. Go to dash.cloudflare.com and log in
    2. Click on the domain you want to configure
    3. On the Overview page, scroll down the right sidebar
    4. Find "API" section — copy the Zone ID

    2 Create an API Token

    1. Go to dash.cloudflare.com/profile/api-tokens
    2. Click "Create Token"
    3. Choose "Create Custom Token"
    4. Set the Token name (e.g., "Luzerge - domainname.com")
    5. Add these permissions:
      • Zone → Cache Purge → Purge
      • Zone → Zone → Read
      • Zone → DNS → Read
      • Zone → Analytics → Read
    6. Under Zone Resources, select "Specific zone" → pick the domain
    7. Click "Continue to summary" → "Create Token"
    8. Copy the token immediately — you won't see it again!
